Overview

Properties

customDelegate

The custom delegate set for the text field. Since DPTextField uses a custom internal delegate, a custom delegate cannot be accessed via the delegate property; use this property instead. You still use setDelegate: to set this delegate.

@property (readonly, nonatomic) id<UITextFieldDelegate> customDelegate

Discussion

Note: Consider using DPTextField’s block methods, instead of a delegate.
